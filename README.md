## üß≠ Storyboard

Berikut adalah gambaran storyboard dari aplikasi **GO-FLASH**, mulai dari pengguna membuka aplikasi hingga melakukan proses pemesanan jasa kurir:

1. **Splash Screen**
   - Tampilan logo GO-FLASH selama 2 detik
   - Dilanjutkan ke halaman login atau home (jika sudah login)

2. **Login / Register**
   - Pengguna bisa masuk menggunakan email/password atau Google Sign-In
   - Jika belum punya akun, bisa mendaftar terlebih dahulu

3. **Beranda (Dashboard)**
   - Tampilan peta dan lokasi pengguna saat ini
   - Tombol utama: "Kirim Paket", "Riwayat", dan "Akun Saya"

4. **Form Pengiriman**
   - Input lokasi asal & tujuan
   - Pilih jenis kendaraan kurir (motor, mobil, pickup)
   - Estimasi ongkos kirim ditampilkan real-time

5. **Konfirmasi & Pembayaran**
   - Ringkasan pengiriman
   - Metode pembayaran: e-wallet, COD, atau transfer

6. **Tracking Real-time**
   - Setelah order dikonfirmasi, pengguna bisa melacak posisi kurir secara langsung

7. **Riwayat & Ulasan**
   - Daftar pengiriman sebelumnya
   - Pengguna bisa memberikan rating & ulasan
